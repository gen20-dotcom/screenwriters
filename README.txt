SCREENWRITER DESKTOP APP FOR WINDOWS

This is the Windows desktop version of your HTML/CSS/JS screenwriting editor.
It uses Electron so the same website becomes an installable Windows app.

WHAT IS INCLUDED
- Home screen like Google Docs
- Blank screenplay
- Import .swscript / .fountain / .txt / .json
- Recent documents using local storage
- Smart slugline suggestions
- Character autocomplete
- Transition suggestions
- Sidebar scene reorder
- Title page settings
- Save PDF export
- Scene heading bold

HOW TO RUN FOR TESTING
1. Install Node.js LTS on Windows.
2. Extract this ZIP.
3. Double-click run-windows.bat.

HOW TO CREATE WINDOWS INSTALLER
1. Extract this ZIP.
2. Double-click build-installer-windows.bat.
3. Wait for build to finish.
4. Open the dist folder.
5. Install Screenwriter Setup.exe.

HOW TO CREATE PORTABLE APP
1. Double-click build-portable-windows.bat.
2. Open the dist folder.
3. Use the portable .exe.

NOTES
- Your scripts are stored locally on your computer.
- Use Export > Screenwriter File (.swscript) to save an editable project file.
- Use Export > Save PDF for a clean PDF without browser headers/footers.
